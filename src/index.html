I am template for a project that uses npm, webpack, eslint, babel and jest among others.See package.json for details

After downloading this project be sure to update my 'name' and 'description' properties in package.json to reflect the new project

I also contain an empty .env file which can be used thanks to "dotenv-webpack": "^2.0.0". .env should be deleted and the dependency removed from package.json if not being used.

From the command line 'npm install', then 'npm run start' to install all dependencies and start a live server
