• I am template for a project that uses npm, webpack, eslint, babel and jest among others.See package.json for details
  • I include bootstrap and jquery
  • After downloading this project be sure to update my 'name' and 'description' properties in package.json to reflect the new project
  • If you do not intend to use a .env file he dependency "dotenv-webpack": "^2.0.0" should be removed from package.json
  • If you do not intend to use jQuery in your project the dependency "jquery": "^3.5.1" should be removed from package.json
  • If you do not intend to use Bootstrap in your project the dependencies "bootstrap": "^4.5.0" and "popper.js": "^1.16.1" should be removed from package.json
  • From the command line 'npm install', then 'npm run start' to install all dependencies and start a live server
